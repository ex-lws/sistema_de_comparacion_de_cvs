Seguir los siguientes pasos para poder trabajar en el mismo código. 

1.- git pull = actualiza el repositorio clon a los últimos cambios del repositorio remoto.
2.- Tras realizar y probar cambios, dar click en el ícono de + (stage all changes).
3.- Crear un commit bien desciptivo con todos los cambios que queremos subir al repositorio remoto.
4.- git push = sube los cambios oficialmente al repositorio remoto.

Para poder clonar un repositorio.

1.- Pedir la direccion HTTPS o SSH al administrador del repositorio en GitHub.
2.- Abrir Git Bash y mediante comandos Linux entrar a la carpeta que contendra el proyecto. 
3.- User git clone DireccionHttpsDelRepositiorio.
